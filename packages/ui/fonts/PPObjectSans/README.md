# PP Object Sans Font

This directory should contain the PP Object Sans font files.

## Required Files

Place the following font files in this directory:

- `PPObjectSans-Regular.woff2`
- `PPObjectSans-Medium.woff2`
- `PPObjectSans-Bold.woff2`

## Usage

The font is automatically loaded by consuming applications using `next/font/local` or similar font loading mechanisms.

## License

PP Object Sans is a commercial font. Ensure you have the proper license before using it in your projects.

## Fallback

If the font files are not available, the system will fall back to the default sans-serif font stack.
